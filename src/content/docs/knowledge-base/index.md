Long-form, searchable articles for troubleshooting, setup, and best practices.

## For Contributors

Want to add a new knowledge base article?

1. Run `yarn new:kb <category> "Article title"` (e.g. `yarn new:kb Avatar "Broken avatar"`)
2. Fill in the generated draft file under `src/content/docs/knowledge-base/<category>/`
3. Place any screenshots in the created `images/<entry-slug>/` folder
4. Commit and open a PR

You can also copy from the template in `src/content/docs/knowledge-base/template.md`.
