## For Contributors

Want to add a new knowledge base article?

1. Run yarn new:kb "Article title"
2. Fill in the generated draft file under src/content/docs/knowledge-base
3. Commit and open a PR

You can also copy from the template in src/content/docs/knowledge-base/template.md.
