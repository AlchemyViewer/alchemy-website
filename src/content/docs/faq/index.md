Quick answers to common questions.

## For Contributors

Want to add a new FAQ entry?

1. Run `yarn new:faq <category> "Your question"` (e.g. `yarn new:faq Settings "How do I...?"`)
2. Fill in the generated draft file under `src/content/docs/faq/<category>/`
3. Place any screenshots in the created `images/<entry-slug>/` folder
4. Commit and open a PR

You can also copy from the template in `src/content/docs/faq/template.md`.
